Cómo publicar rápido

  1. Guarda este archivo como index.html.
  2. Sube a GitHub (repositorio) o arrástralo a Netlify/Vercel/Cloudflare Pages.
  3. Si usas GitHub Pages: configura “Pages” y conecta tu dominio con registros DNS.
